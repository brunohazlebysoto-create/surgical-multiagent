# Quirófano Virtual Pediátrico

Sistema multi-agente de consenso quirúrgico para cirugía pediátrica. Busca papers en PubMed, Semantic Scholar, CrossRef y OpenAlex, analiza la evidencia con marco PICO-S y GRADE, y genera apuntes clínicos (.docx) y presentaciones (.pptx).
